Page 布局

h5kit 是一套基于 Vue2.0 的 SPA 快速开发的UI组件库。每个视图对应一个Page，整个Page可以分为header、content、footer三个组件。

header 组件

header 组件是提供整个 Page 的头部，按常规设置默认样式高度为 44px 。从功能上默认分为“回退”、“标题”、“按钮”，其中“回退”和“按钮”可以自定义，标题的文本可以配置。

基本用法：

<v-header title="首页" btn-txt="确定"></v-header>
  • title 是可以配置的标题文本。
  • btnTxt 是可以配置的按钮文本。

自定义“回退”

  • 嵌入HTML代码

    <v-header title="首页">
  <a href="#" slot="regret" class="regret yo-ico custome">&#xe604;</a>
</v-header>

  • 嵌套组件

    <v-header title="首页">
  <regret slot="regret"></regret>
</v-header>

自定义“按钮”

  • 嵌入HTML代码

    <v-header title="首页">
  <a href="#" slot="affirm" class="affirm yo-ico custome">&#xe604;</a>
</v-header>

  • 嵌套组件

    <v-header title="首页">
  <affirm slot="affirm"></affirm>
</v-header>

事件

  • 回退

    如果“回退”没有自定义，默认是处理了回退事件的，开发者可以不用关心

  • 按钮

    如果“按钮”没有自定义，默认提供了 affirm 事件函数，开发者可以传参使用。

    <v-header title="首页" btn-txt="确定" :affirm="ok"></v-header>

    其中ok是引用页面中methods中定义的方法。

content 组件

content 组件提供一个可伸缩的容器，配合 header 组件和 footer 组件完成常规的 Page 的布局（上中下）。

使用方法：

<v-content>
  <v-list>
    <v-item v-for="n in 24"></v-item>
  </v-list>
</v-content>

其中 v-list 是嵌套的其他组件。

footer 组件

footer 组件是 App 中常见的底部 Tab 导航。

使用方法：

<v-footer>
  <template v-for="item in footer">
    <footer-item :icon="item.icon" :text="item.text" :active="item.active" :link="item.link"></footer-item>
  </template>
</v-footer>
  • icon 是图标编码
  • text 是文本
  • active 是否激活
  • link 路由选项

如果自定义可以替换 footer-item 组件，不过不建议自定义。
